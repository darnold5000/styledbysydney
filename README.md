# Styled by Sydney

Personal one-page booking site for Sydney — a hair stylist at Bloom Studio Salon in Avon, Indiana.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Configuration

Edit `src/config/stylist.ts` to update:

- Brand name and domain
- Booking URL (Square, GlossGenius, Vagaro, Calendly, etc.)
- Contact info and social links
- Services, gallery images, and copy

## Deploy

Built for [Vercel](https://vercel.com). Connect the `styledbysydney` repo and deploy.

Set `bookingUrl` in config before launch.

## PWA

The site includes `manifest.ts`, app icons, and add-to-home-screen instructions so clients can save it like a lightweight booking app.
